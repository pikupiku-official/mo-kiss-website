# 画像配置ガイド

このドキュメントでは、モーキス公式サイトで使用する画像の配置場所と推奨サイズを説明します。

## ディレクトリ構成

```
images/
├── logo/              # ロゴ画像
├── characters/        # キャラクター画像
├── icons/             # アイコン・装飾画像
│   └── counter/       # カウンター数字画像（自動生成）
└── backgrounds/       # 背景画像
```

## 必要な画像

### 1. キャラクター画像

**配置場所**: `images/characters/`

**ファイル名例**:
- kanna.png (烏丸神無)
- misuzu.png (桔梗美鈴)
- momoko.png (愛沼桃子)
- sanako.png (舞田沙那子)
- miyori.png (宮月深依里)
- kurenai.png (伊織紅)

**推奨サイズ**: 幅180px〜300px、高さ自由
**形式**: PNG、JPG、GIF

### 2. ロゴ画像

**配置場所**: `images/logo/`

**ファイル名例**:
- logo_main.png (メインロゴ)
- logo_title.png (タイトルロゴ)

**推奨サイズ**: 幅300px〜600px
**形式**: PNG（透過推奨）

### 3. アイコン画像

**配置場所**: `images/icons/`

**必要なアイコン**:
- new.gif (NEWアイコン、40x15px程度)
- mail.gif (メールアイコン、40x15px程度)
- construction.gif (工事中アイコン、任意サイズ)

**形式**: GIF（アニメーションGIF推奨）

### 4. 背景・装飾画像

**配置場所**: `images/backgrounds/`

**用途**:
- 背景パターン
- 区切り線
- 星・宝石などの装飾

**形式**: PNG、GIF

## レトロアイコンの作成方法

1999年風のアイコンを作成する場合、以下の特徴を意識してください：

### NEWアイコンの例
- 背景色: 赤、黄色、オレンジなど原色系
- 文字: 白または黄色で「NEW」
- サイズ: 40x15px程度
- アニメーション: 点滅や色変化

### MAILアイコンの例
- 封筒のイラスト
- 背景色: 青、緑など
- サイズ: 40x15px程度

### 工事中アイコンの例
- 工事標識や「工事中」の文字
- 黄色と黒のストライプ
- サイズ: 50x50px程度

## 画像のアップロード方法

1. FTPクライアントまたはサーバーのファイルマネージャーを使用
2. 該当ディレクトリに画像をアップロード
3. 管理画面でパスを指定（例: `images/characters/kanna.png`）

## 注意事項

- ファイルサイズは1枚あたり500KB以下を推奨
- ファイル名は半角英数字のみを使用
- 著作権に注意し、使用許可のある画像のみ使用
- キャラクター画像は縦長を推奨（立ち絵風）

## カウンター画像について

カウンター画像（0〜9の数字）は`api/counter.php`で自動生成されるため、
手動で配置する必要はありません。

自動生成される画像の仕様:
- サイズ: 20x25px
- 背景: 黒
- 文字色: 赤
- 枠線: 金色
